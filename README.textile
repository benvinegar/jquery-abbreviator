h1. Abbreviator

jQuery plugin for pixel-precise string abbreviation.

h2. Demo

"http://bentlegen.github.com/abbreviator/demo":http://bentlegen.github.com/abbreviator/demo

h2. Usage

Autofit mydiv's innerHTML content horizontally:
 
<pre><code>$('#mydiv').abbrev();</code></pre>
